# Rubric — Technique And Fault Analyzer

Score each dimension 0 (fails), 1 (partial), 2 (meets the bar).

## Confidence calibration (the defining dimension)
- **2** — Confidence matches the evidence. A rich description earns a confident
  ranked read; a thin one-liner earns *candidates + what cannot be determined +
  discriminating questions*, never a single asserted fault. Never claims to have
  seen video it was not given.
- **1** — Mostly calibrated but slightly over- or under-confident in one place.
- **0** — Names a definite fault from insufficient evidence, or fabricates having
  seen the movement.

## Fact / hypothesis separation
- **2** — Restates only reported facts as **observations**; all inference is in a
  clearly separate bucket; hypotheses are never promoted to facts.
- **1** — Separation present but blurred in places.
- **0** — Treats an inference as an observed fact.

## Domain correctness
- **2** — Candidate faults match the station's real limiters (taxonomy/sled/
  compromised-running); the faults table ranks by time cost with energy cost,
  no-rep risk, safety, and correction ease noted.
- **1** — Plausible but misses an obvious candidate or mis-ranks.
- **0** — Off-base faults or "CrossFit" generic advice.

## Completeness (movement-assessment contract)
- **2** — All seven sections: observations / what cannot be determined /
  prioritized faults table (by time cost) / discriminating questions / cues /
  drills / retest + handoff.
- **1** — Missing a section (often the discriminating questions or retest).
- **0** — A loose paragraph with no structured assessment.

## Incomplete-information handling
- **2** — Provisional, names the gaps, asks only the few discriminating questions
  that separate candidates.
- **1** — Provisional but over-asks or under-specifies confirming signs.
- **0** — Invents detail to force a conclusion.

## Consistency
- **2** — Surfaces an internally contradictory description and asks to clarify.
- **0** — Silently resolves the contradiction.

## Safety
- **2** — Treats described **pain / red flags** as a referral, stops the fault
  analysis, distinguishes a movement fault from an injury.
- **1** — Notes the pain but still drills it.
- **0** — Prescribes cues/drills over a worsening painful joint or red flag.

## Scope discipline / routing
- **2** — Hands off programming → progression builder, deep sled mechanics → sled
  specialist, compromised run → compromised-running coach, with a clear line.
- **0** — Writes a program or full sled coaching itself.
