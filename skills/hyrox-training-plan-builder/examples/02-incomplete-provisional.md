# Example 02 — Incomplete data: provisional plan with a short clarifying ask

## Request

> First HYROX in about 3 months. Can you build me a training plan?

## Response (excerpt)

I can give you a **provisional** first-timer plan now and tighten it with two quick
answers. I'm not refusing — I'm flagging what's assumed so you can trust the numbers.

**What materially changes the plan (please confirm):**

1. **Days/week and session length** you can realistically train.
2. **Equipment access** — do you have a sled, an erg (ski/row), and a wall-ball
   target, or gym/treadmill only?

I do **not** need a goal time or exact paces to start — for a first race, finishing
strong is the goal, and we'll set paces from a week-1 test.

**Provisional architecture (assumed: ~12 weeks, 4 days/week, ~60 min, full gym —
all flagged).**

| Phase | Weeks | Objective |
|-------|-------|-----------|
| General prep | 1–4 | Easy running base, learn all 8 stations safely, week-1 test |
| Capacity | 5–8 | Threshold runs, station conditioning, first station-to-run bricks |
| Race-specific | 9–11 | Goal-pace bricks, one partial simulation in week 11 |
| Taper | 12 | Volume down, intensity kept → race-week skill takes over |

**Provisional weekly skeleton (assumed 4 days):** 2 easy runs + strides, 1 threshold
or brick session, 1 strength/station session. Easy pace set conversationally
(RPE 3–4) until the week-1 test gives a number — I will **not** invent a pace for an
athlete I haven't tested.

**Progression:** add one easy run or +10% easy volume only after two comfortable
weeks; introduce bricks only once each station is movement-competent.

**Assumptions flagged:** weeks (≈12), days (4), session length (60 min), and full
equipment are all assumed; confirm them and I'll lock the plan. If you have only a
treadmill, the station work becomes substitutions — that's a different weekly shape
(`hyrox-equipment-substitution-engine`).

**Scope note:** coaching, not medical advice; any pain that is sharp or worsening
routes to [safety-and-referral](../../../references/safety-and-referral.md).
