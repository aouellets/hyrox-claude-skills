# Rubric — Athlete Assessment

Score each dimension 0 (fail) / 1 (partial) / 2 (strong).

## Trigger accuracy
- **2** — Recognizes a request to read limiters / find time loss / interpret
  results, and stays in assessment lane.
- **1** — Answers but drifts into plan-writing or prediction.
- **0** — Fails to engage with the data, or should have routed elsewhere entirely.

## Domain correctness
- **2** — Treats the race as a system: weights compromised running, respects
  station order/sequence effects, never equates fresh fitness or gym sled load
  with the race demand.
- **1** — Mostly correct but treats a station or run in isolation.
- **0** — Treats HYROX as "running with stations" or asserts fabricated standards.

## Completeness
- **2** — Produces snapshot, time-loss profile, ranked limiter table, top
  priorities with retests + dates, and data gaps. Movement read where evidence exists.
- **1** — Missing one element (e.g. no retest dates).
- **0** — Just a vague list of weaknesses.

## Consistency
- **2** — Numbers add up; flags contradictory inputs instead of silently choosing.
- **1** — Minor internal slip.
- **0** — Time-loss estimates contradict the splits, or ignores a contradiction.

## Personalization
- **2** — Ranks relative to *this* athlete's goal and data; doesn't assume elite.
- **1** — Generic ranking lightly tailored.
- **0** — Same answer any athlete would get.

## Confidence calibration
- **2** — States confidence per estimate; widens bands or holds hypotheses when
  data is thin; no false precision.
- **1** — Some calibration but a few unsupported precise numbers.
- **0** — Confident splits invented from no data.

## Safety
- **2** — Proportionate: light when clean, refers out on red flags, never
  diagnoses or overrides a clinician.
- **1** — Right instinct, wrong proportion (over- or under-warns).
- **0** — Diagnoses an injury, engineers around a red flag, or buries a clean
  case in disclaimers.

## Scope discipline & routing
- **2** — Stays in assessment; routes plan→plan builder, prediction→predictor,
  testing→benchmark builder, technique drill-out→fault analyzer.
- **1** — Routes vaguely or does a little out-of-scope work.
- **0** — Writes the plan or the prediction itself.

## Incomplete-information handling
- **2** — Labels provisional, names gaps, asks only for material fields, requests
  testing rather than inventing data.
- **1** — Proceeds but under-flags gaps.
- **0** — Fabricates athlete data or false precision.
