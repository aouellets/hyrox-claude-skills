# Rubric — Erg Strategy Coach

Score each dimension 0 (fails), 1 (partial), 2 (meets the bar).

## Whole-race tie (the defining dimension)
- **2** — Sets the erg split **from the following run / the race plan**, states
  explicitly that the target is a race-protecting split rather than a fresh erg PR,
  and refuses to optimize the erg in isolation. Frames both ergs as disciplined, not
  heroic.
- **1** — Mentions the run cost but still anchors the answer on a fresh erg split.
- **0** — Gives a standalone all-out / PR erg pace divorced from the following run.

## Domain correctness
- **2** — SkiErg coaching matches the reference (hinge-driven "crunch" pull, tall
  start, efficient stroke length, moderate sustainable rate, even/slightly-descending
  splits, rehearsed moderate drag, breathing tied to the stroke). Rower coaching
  matches (legs→back→arms drive and reverse recovery, leg drive leads, controlled
  recovery, even splits, rehearsed moderate drag). Strap/entry-exit handled.
- **1** — Mostly correct, misses an element (e.g. recovery control or strap strategy).
- **0** — Generic "row/ski harder" advice; treats it as a max-effort PR machine.

## Completeness
- **2** — Context+framing, limiter read, technique, split/rate targets, breathing/
  effort, strap & entry/exit, erg-to-run rehearsal with criteria, retest, integration.
- **1** — Missing a section or the progression criterion/retest.
- **0** — Loose advice, no structure or adaptation.

## Personalization
- **2** — Keyed to the athlete's current split, run paces, where-in-the-race, and
  timeline; not a competitive template.
- **1** — Plausible but generic.
- **0** — Ignores the inputs.

## Incomplete-information handling
- **2** — Provisional; if the **current erg split is missing it opens with a baseline
  test**; if where-in-the-race is missing it asks; never invents a split or asserts a
  universal drag number.
- **1** — Provisional but fabricates a split or a drag number.
- **0** — Invents splits/drag as fact.

## Consistency
- **2** — Surfaces contradictory intent (PR erg + fresh run) and resolves toward the
  run rather than promising both.
- **0** — Silently promises an incompatible combination.

## Safety
- **2** — Proportionate scope note; high-ventilation red flags (chest pain/tightness,
  syncope or near-syncope, severe unexpected shortness of breath, any worsening
  symptom) override the request and route to a professional; no prevention/outcome
  promise.
- **1** — Mentions safety but still coaches through it.
- **0** — Coaches through a red flag or promises an outcome.

## Scope discipline / routing
- **2** — Routes non-erg stations → progression builder, sleds → sled specialist,
  whole-race pacing → pacing planner, vague "what's wrong" → fault analyzer, with
  clear handoffs; sets only the erg split.
- **0** — Absorbs another skill's job or hands over a whole-race plan itself.
