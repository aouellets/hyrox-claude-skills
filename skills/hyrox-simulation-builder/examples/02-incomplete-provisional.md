# Example 02 — Incomplete data (provisional half-simulation)

**Request:**

> "Can you build me a HYROX simulation for this weekend?"

## Provisional — this is a labeled best-guess, not a finished prescription

I can give you a sensible default, but a few unknowns change the format. Here's a
**provisional half-simulation** built on stated assumptions; confirm the items at
the bottom and I'll sharpen it.

### Assumptions (stated, not invented)

- You're mid-block (not in race week or a taper) and not currently niggling or ill.
- You have access to most stations or close substitutes.
- Singles format, recreational-to-competitive goal, no HR file.
- Fresh 1 km pace ~5:30/km (a placeholder — I have **not** assumed your splits).

If you're inside 10 days of a race, under-recovered, or coming back from illness,
this is the wrong week for it and I'd swap to easy bricks instead.

## Format chosen: half / segmented (not a full sim)

A **half simulation — 4 run-station pairs** in race order (Run → SkiErg → Run →
Sled Push → Run → Sled Pull → Run → Burpee Broad Jumps). This tests pacing and
accumulated fatigue at meaningful but recoverable cost. I'm **not** defaulting to
a full sim on missing data — that's the costly wrong answer.

### Session

- ~1 km run before each station at planned race effort; stations at repeatable
  race effort (not to failure).
- Label any substitution (e.g. a treadmill for outdoor runs, a comparable carry if
  a station is missing) so the data carries the caveat.

### Intensity

Race effort, RPE building from ~7 early to ~8 late; first 100–200 m out of each
station deliberately controlled.

### Data to collect

Per-km run paces (fresh and compromised), station times, transition times, RPE,
grip/technique failures, subjective notes.

### Recovery cost

Moderate-to-high; plan an easy day either side.

### Advancement criterion

When pacing holds across all four pairs and the fade is in your planned band,
you've earned a single, sparse full simulation later in the build.

## What would sharpen this (tell me these)

1. How many days until your race? 2. Any niggle, illness, or recent layoff?
3. Which stations/equipment do you actually have? 4. A recent split or fresh 1 km
pace so I stop using a placeholder. 5. Singles, doubles, or relay?
