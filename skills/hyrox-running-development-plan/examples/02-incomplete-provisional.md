# Example 02 — Incomplete data: provisional block, anchor-test first

## Request

> Can you build me a running plan for HYROX? I can run maybe 3 times a week. I
> don't really know my paces — I just run by feel. Race is "sometime this autumn."

## Response (provisional — labeled)

**This is a provisional plan.** I have your availability (3 runs/week) but no
objective running fitness and no firm race date, so I will not invent paces or a
precise phase count. The first session anchors the rest.

**What I'm missing and why it matters:**

| Gap | Effect on the plan |
|---|---|
| Current running fitness (a time/test) | Sets every working pace — easy, threshold, race-pace. Without it, paces are by-feel only. |
| Firm race date | Sets the number of weeks and the phase emphasis. "Autumn" is too wide to phase. |

**Anchor test (do this first, then I'll set real paces).**
A paced effort you can do solo:

- 1 km time trial at a hard-but-repeatable effort (RPE ~8), fully warmed up; OR
- a 3 km steady run you could *just* sustain, noting average pace.

Report the result and I'll derive easy / threshold / race-pace from *your* number.

**Provisional structure (by RPE until paces exist), 3 runs/week:**

- **Run A — easy aerobic.** 35–45 min conversational, RPE 3–4. Easy stays easy.
- **Run B — threshold.** After 2 base weeks: 4 × 4 min "comfortably hard" (RPE
  7–8), 90 s jog. Progress to 5 × 4 → 6 × 4 when the prior session felt repeatable.
- **Run C — easy + strides.** 30–40 min easy + 6 × 20 s relaxed fast strides.

**Impact note.** With 3 runs/week and unknown history, the ramp is conservative:
extend the easy run ~10%/wk max, no quality the day after a heavy lift, a down
week every 4th week.

**To upgrade this to a real plan**, send: the anchor-test result and a firm race
date (or weeks-to-race). I'll then phase the block and assign paces. If you also
want stations and strength in one program, route to
`hyrox-training-plan-builder`.
