# Example 03 — Routing + safety: post-station fade vs. shin pain

## Request A — wrong skill for the ask (route)

> Build me a running block. My problem is the run right after the sleds — my legs
> are dead and I lose tons of time in the first 200 m out of the roxzone.

### Response

That is a **compromised-running** problem, not a running-capacity one. This skill
builds the engine (base, threshold, race-pace, durability); the skill that trains
*recovering turnover and pace in the first 100–200 m off heavy legs after a
station* is `hyrox-compromised-running-coach`. The fix you're describing —
cadence restoration, breathing recovery, pace control out of the sled, sled-to-run
bricks — lives there.

The two usually run together: I can build the aerobic and race-pace base here
**and** route the sled-to-run transition work to the compromised-running coach so
they're sequenced (heavy run engine on easy days, bricks layered in as the block
matures). Want both? If so I'll need your current running fitness, running days,
and race date to build my half. For the whole program in one block, route to
`hyrox-training-plan-builder`.

## Request B — niggle, then a red flag (safety)

> 12 weeks out, can run 4x/week, 5 km is 23:00. Build it. One thing — I get a dull
> ache along my left shin after longer runs, eases with rest. Also last week I had
> a sharp, pinpoint pain on the shin bone that's getting worse each run and is now
> sore even walking.

### Response

I need to flag the second symptom before any plan.

**A sharp, pinpoint pain on the shin bone that is worsening run to run and now
hurts on walking is outside coaching scope** and is a pattern that warrants
prompt evaluation by an appropriate medical professional (it should be assessed
to rule out a bone-stress problem). I will not program around it or tell you to
"run through it." Please get it looked at before loading the leg further.

I can't responsibly hand you a 4-day running build while that symptom is active —
adding mileage is the wrong move here. Once you've been cleared and the pinpoint
pain has resolved, I'll build the block, and I'd start the running ramp
conservatively and watch the dull post-run ache (keep load gentle, regress if it
worsens, stop and re-seek advice if the sharp pain returns).

This is coaching guidance, not a medical opinion — the diagnosis and clearance are
your clinician's call, not mine.
