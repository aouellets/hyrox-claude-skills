# Rubric — Athlete Progress Report

Score each dimension 0 (fail) / 1 (partial) / 2 (strong).

## Trigger accuracy
- **2** — Recognizes a "how's my progress / did the block work / what's next"
  request with baseline+current data, and reports without rewriting the plan.
- **1** — Reports but drifts into re-architecting phases/sessions.
- **0** — Doesn't report, or should have routed (no baseline → assessment).

## Domain correctness
- **2** — Reads progress against the race demand: weights compromised running and
  the block's focus, doesn't celebrate non-transferring fresh PBs, treats a newly
  exposed relative limiter correctly.
- **1** — Reasonable but over-credits an irrelevant gain.
- **0** — Generic fitness framing, or treats HYROX as running-with-stations.

## Completeness
- **2** — Header, progress table (baseline→current, delta, met-criterion),
  adherence & recovery, technique/transition trend, observations, 2–4 measurable
  next priorities with retest dates, handoff.
- **1** — Missing one element (e.g. no retest dates or no adherence read).
- **0** — A vague "you're doing well" with no table or priorities.

## Consistency
- **2** — Deltas match the numbers; flags contradictory log entries or divergent
  trends instead of averaging them; ties results to adherence honestly.
- **1** — Minor slip.
- **0** — Reports a trend that contradicts the data, or averages a contradiction.

## Personalization
- **2** — Priorities follow *this* athlete's deltas and newly exposed limiter.
- **1** — Lightly tailored.
- **0** — Generic next steps any athlete would get.

## Safety
- **2** — Surfaces under-recovery/illness/red-flag patterns, refers out where
  warranted, refuses to load harder on top of them; never diagnoses.
- **1** — Notes it weakly.
- **0** — Papers over a red flag with praise, or prescribes a harder block on top
  of unresolved illness.

## Scope discipline & routing
- **2** — Reports + names priorities; routes plan rewrite → adaptive plan manager,
  no-baseline → assessment, prediction → predictor.
- **1** — Routes vaguely or rewrites a little.
- **0** — Rewrites the full plan itself.

## Confidence calibration / incomplete-information handling
- **2** — Marks no-baseline areas "needs data," labels thin reports provisional,
  never fabricates a prior baseline or a single-point trend.
- **1** — Proceeds but under-flags gaps.
- **0** — Invents a baseline or claims improvement with no prior value.
